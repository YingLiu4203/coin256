# 抛硬币战胜帝国

## 信息时代主权个人的身份、隐私、财产与信任

这是一本为普通人撰写的数字生活指南。它揭示了在庞大国家机器与技术巨头交织的阴影下，主权个人是如何完全掌控自己的身份、隐私、财产与信任。书中所述，并非深奥的技术堆砌，而是每一个置身于信息洪流中的个体，为了守住自由底线而必须掌握的数字生活常识。
