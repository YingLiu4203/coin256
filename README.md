# 拋256次硬幣就可獲得資訊與財產自由

> 現代人的數位生活常識

這是一本寫給普通人的書，是關於個人面對強大的政府機構和技術公司時如何保護個人的信息自由與財產自由這二種基本人權的書。書中內容是資訊時代人們應該具備的數位生活常識。

## 關於書名

如果一句話可以概括整本書的內容，那麼用這句話來作書名，應該不算太長。畢竟看到書名就看到了整本書的結論，看過書的或懂得其中道理的就會又一次認識到個人最寶貴的信息和財產自由竟然如此簡單就可以得到，即使最強大的帝國調動其所有資源也無法奪走屬於你的信息和財產的主權、交易分享權和隱私權。
