# 抛硬币战胜帝国：SovinWeb 时代的主权个人指南

SovinWeb，即新一代主权个人互联网。它标志着权力的彻底回归，旨在由你亲手掌握数据、通信与财产的绝对主权。

![cover](./content/imgs/cover.png)

在技术巨头与庞大国家机器交织的阴影下，这是一本为普通人撰写的“数字独立宣言”。它旨在揭示：如何以密码学与主权个人信息系统为盾，捍卫数字时代最核心的两大主权——言论的自由流动与财产的绝对掌控。

书中所述，并非深奥的技术堆砌，而是每一个置身于信息洪流中的个体，为了守住自由底线而必须掌握的生存常识。
