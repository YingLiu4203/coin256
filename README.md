## 关于书名

如果一句话可以概括整本书的内容，那么用这句话作书名，应该不算太长。毕竟看到书名就看到了整本书的结论，看过书的或懂得其中道理的就会又一次认识到个人最宝贵的信息和财产自由竟然如此简单就可以得到，即使最强大的帝国调动其所有资源也无法夺走属于你的信息和财产的主权、交易分享权和隐私权。
