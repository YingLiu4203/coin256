# 7-5 Web3 迷雾

这个俄罗斯套娃最外层、最大、也最空洞的外壳，正是 Web3。这是一个极具诱导性的概念，试图将区块链包装成互联网不可避免的下一次进化。Web3 宣称要打造一个可读（Read）、可写（Write）且可拥有（Own）的网络，赋予用户真正的数字所有权。但现实中，它不过是一个脆弱的寄生外壳，笨拙地寄生在现有的 Web 基础设施之上。这根本不是什么技术革命，而是一场移花接木的叙事戏法。

## 1 来历

Web3 的历史就是一部身份窃取与语义重塑的变迁史。这个标签在过去二十年里经历了三次彻底的定义重构，每一次都折射出当时技术圈不同的困境与野心。

Web 1.0 是互联网的拓荒时代，它是一个只读的巨型图书馆，信息流向是单向的，用户仅仅是静默的消费者。到了 21 世纪初，Web 2.0 时代开启了读写互动的变革，用户开始成为内容的创造者，社交媒体和博客让每个人都能发声。然而，Web 2.0 的繁荣是以高度中心化为代价的，用户创造的价值和数据最终被困在少数科技巨头的围墙花园里。正是在这种背景下，人们开始构思互联网的下一个形态，并赋予了它 Web 3.0 的标签。

第一版 Web 3.0 的概念早在 1999 年就已萌芽。万维网的发明者蒂姆·伯纳斯-李将其定义为语义网。在那个人们对人工智能还充满朴素幻想的年代，伯纳斯-李设想了一个机器可读的互联网。他认为 Web 2.0 的数据是无序的，机器无法理解文本背后的含义。他憧憬一个通过元数据互联的网络，使得智能代理能够代表人类自动完成复杂的任务，比如订票或研究课题。需要指出的是，这虽然是一个由于架构过于沉重且缺乏商业动力而最终停滞不前的设想，但它是一次极其严肃的技术主权与效率的探索。令人遗憾的是，这一标签后来被投机者盗用，成了完全背离其技术初衷的营销辞令，沦为这一俄罗斯套娃结外层被透支的技术名词。

第二版 Web 3.0 诞生于 2014 年。当时，爱德华·斯诺登揭露的美国全球监控丑闻震撼了世界，人们开始对中心化互联网巨头产生极度的不信任。以太坊联合创始人加文·伍德在此时的一篇博文中，将 Web 3.0 重新定义为安全的社交操作系统。伍德的愿景深受密码朋克文化影响，他提出要从不作恶转变为不能作恶。他设想了一个无中心服务器的互联网，利用加密证明取代中介机构的信用。但这个概念局限在极小众的技术社区内，并未引起大众市场的关注。

第三版 Web3，即我们今天所熟知的版本，大约在 2021 年被精心推向台面。这是一个具有高度策略性的品牌重塑过程。当时，加密货币和区块链这二个词由于与勒索软件、庞氏骗局和环境破坏等负面新闻紧密相连，已经引起了公众的普遍反感。为了掩盖这些污名，以 a16z 为首的一众硅谷顶级风险投资机构开始积极推广 Web3 这一术语。

这是一次绝妙的叙事转移：通过提出“读、写、拥有”（Read-Write-Own）的口号，将一个充满争议的金融资产类别转变为一种旨在夺回互联网权力的道德叙事。在这个叙事模型中，Web3 被定义为互联网进化的下一步，而其核心灵魂就在于拥有（Own）。这一版 Web3 宣称通过区块链代币和加密算法，用户可以真正拥有自己的数字身份、内容和资产，而不是向平台租用。其核心逻辑就是将所有权（Ownership）技术化和绝对化。它承诺用户可以拥有互联网的一部分，并将原本枯燥的技术术语包装成了赋能个体的自由宣言。它巧妙地将人们对比特币个人主权理念的渴望，嫁接到了各种需要大量资本推动的、中心化色彩浓厚的新概念产品上。

但那些所谓的创新，要么是披着去中心化皮的传统数据库，要么是因效率低下而无法维持基本运营的死胎。事实上，这一宏大叙事下的区块链应用无一不在现实的商业逻辑和用户需求面前败下阵来，没有任何一个项目在现实世界中取得真正意义上的成功。

## 2 Web 3 愿景

Web3 的定义是基于互联网演进的三个阶段来描述的：Web 1.0（读）：用户消费由发布者提供的内容（例如，AOL、静态HTML）；Web 2.0（读、写）：用户创建内容（社交媒体、博客），但平台拥有数据、算法和盈利权；Web3（读、写、拥有）：用户创建内容，并通过私钥拥有底层资产。然而，这个定义揭示了 Web3 理念的一个关键缺陷：它过于强调拥有权，却对所有权在现实中如何运作的机制含糊其辞。Web3 的支持者常常将“持有私钥”与“拥有产权”混为一谈，忽略了所有权是一个由法律执行、社会认可和技术效用构成的复杂网络，而不仅仅是去中心化电子表格中的一行数据。Web3 的核心愿景体现在自主身份，数字资产主权与可编程货币这三方面。

### 2.1 自主身份

在 Web3 中，你的身份就是你的钱包地址（如通过以太坊登录），这建立在坚实的公钥加密技术之上。理论上，你可以将自己的声誉、交易历史和社交关系图随身携带，无需经过任何平台许可——这似乎预示着你的粉丝和数字资产将真正归你所有。然而，这种所有权往往只是一种假象。以全球最大的 NFT 交易平台 OpenSea 为例，虽然你确实拥有钱包里的数字藏品，但如果 OpenSea 决定封禁你的钱包地址，你的资产就会瞬间失去流动性与曝光渠道。 系统的接口界面（无论是 OpenSea 还是特定的社交应用）依然握有着生死大权，这也证明了技术上的所有权，并不等同于平台的使用权。

### 2.2 真正的数字所有权和互操作性

Web3 试图将产权引入数字领域。在像 Fortnite 这样的 Web 2.0 游戏中，当你花费 20 美元购买一个角色皮肤时，你仅仅是租用了 Epic Games 封闭数据库中的像素。你无法出售、修改或转移它。在 Web3 中，该皮肤是一种 NFT（非同质化代币）。由于所有权记录存储在公共区块链上，而非私有服务器上，理论上你拥有独立于游戏之外的资产所有权。

但问题在于：你拥有的是收据（代币），而非资产本身（图像或 3D 模型），后者通常存储在标准的 Web 服务器上。如果游戏开发商关闭服务器，你手中仅剩下一份加密安全的收据，而该文件已不复存在。这种所有权完全依赖于它声称要取代的中心化服务的持续存在。

### 2.3 创作者经济与可编程货币

Web3 最诱人的承诺或许在于其对资金流向的重构。在 Web 2.0 流媒体时代，音乐人每次播放只能赚到几美分，而平台却攫取了绝大部分收益。Web3 引入了智能合约，在区块链上的自动执行代码。这使得可编程版税成为可能。音乐人可以将歌曲铸造成代币；代码可以设定为每次代币转售时，售价的 10% 将自动返还给艺术家。这省去了催收机构或律师的麻烦。

然而，这种实现方式存在缺陷：它仅在歌曲交易发生在尊重智能合约的特定生态系统内时才有效。如果用户只是简单地复制音频文件（即盗版），区块链无力阻止，这表明代码即法律远不能替代真正的版权法。

## 3 Web3 现实

经过多年实践结，Web3 成为一个逐渐失去光环的营销术语。其根本问题在于物理定义的效率原则：真正的去中心化极其缓慢且成本高昂。为了让普通用户能够使用这些系统，开发者不得不重新引入他们曾承诺要消除的中心化机制。Web3 并非一个全新的互联网，它只是一个构建糟糕的插件，仍然完全依赖于它声称要推翻的中心化统治者。

为了揭示这种空洞叙事的真实内涵，我们可以通过社交网络、数据存储、所有权控制以及现实资产代币化这四个流行的具体例子来进行深入剖析。

### 3.1 修复社交网络

被誉为 Web3 明星项目的 Farcaster（<https://farcaster.xyz/>），其目标是建立一个足够去中心化的社交协议。它的核心理念是将身份标识与社交内容分离：用户的身份信息存储在区块链上，确保账号的所有权完全由私钥控制；而具体的社交内容，如点赞、发布和关注，则卸载到名为 Hubs 的离线服务器网络中。这种设计的初衷是实现社交主权：即便用户在某个应用前端被封禁，依然能带着自己的社交关系无缝迁移到其他应用上。

然而，这一尝试恰恰揭示了区块链物理限制与社交媒体海量需求之间的不可调和。由于向分布式账本写入数据的速度极慢且成本高昂，Farcaster 无法将每一次交互实时存入链上。为了绕过这一限制，系统被迫采用了一种复杂的混合架构。身份管理留在缓慢但安全的区块链上，而内容的实时验证与同步则依赖于由大量服务器构成的 Hubs 网络。这种分层结构大幅提高了技术准入门槛，普通用户根本无法运行自己的节点。结果是所谓的去中心化社交最终又回到了原点：用户再次受制于单一商业实体的中心化服务。

### 3.2 星际文件系统的存储神话

星际文件系统 IPFS（InterPlanetary File System）听起来像是科幻小说中的产物，它的名字暗示了一个能够跨越行星存储数据的宏大愿景。在 Web3 的叙事中，它被视为去中心化存储的基石，旨在打破科技巨头对云端数据的垄断。为了让这个愿景运转，IPFS 引入了代币激励机制 Filecoin，试图让全球的陌生人贡献出自己硬盘的剩余空间来存储他人的文件。

然而，在实际运行中，这种理想化的存储市场不仅效率低下，而且极其不可靠。与人们习惯的高可靠集中式云存储不同， IPFS 上的文件必须依赖于具体的去中心化互联网物理节点来承载。由于 IPFS 网络是由大量私人志愿者节点构成的，每个节点的存储资源都非常有限。为了防止节点硬盘被海量的临时缓存填满，系统内置了被称为垃圾回收的自动清理机制：节点会自动删除那些仅仅是被动经过、且近期无人访问的内容。如果你上传了一个文件，但没有节点（包括你自己的电脑）通过固化（Pinning）操作声明要永久保留它，这个文件就会像掉入深海的石头一样因节点的空间腾挪而消失。

这就导致了一个滑稽的现状：为了确保数据不被清理，用户不得不转向名为锁定服务的中心化中介。这些服务本质上就是一些由商业公司运营的大型服务器集群，你通过信用卡付钱给他们，让他们替你保留文件。这在操作逻辑上与传统的付费网盘毫无二致，但由于套了一层复杂的分布式协议，它的传输速度更慢，隐私保护也更脆弱。

更具讽刺意义的是访问环节。即便你的文件被成功锁定在某个地方，主流的网页浏览器依然无法直接读取这些地址。为了能让普通用户看到内容，流量必须通过一个名为中心化网关的中转站。这就形成了一个致命的瓶颈：如果这个中转站的运营商收到下架指令或者决定屏蔽某些内容，那么这个所谓的去中心化网络就会瞬间变成一座孤岛。这种架构不仅没有消除中介，反而制造了一个比 Web 2.0 更加脆弱、成本更高的官僚技术体系。

### 3.3 所有权的幻觉

最能够体现 Web3 所有权的幻觉是非同质化代币 NFT 及其背后的 OpenSea。NFT 代表了 Web3 的核心金融愿景。其目标是在互联网上引入数字稀缺性和真正的产权，使用户能够像拥有实体商品一样拥有独特的数字物品，而无需依赖任何中央机构。然而，这一愿景在很大程度上已经停滞不前。继 2021 年的投机狂潮之后， NFT 市场的交易量和公众关注度都出现了灾难性的下滑，这表明其所谓的实用性很大程度上是由虚假交易和投机驱动的，而非真正的技术应用。

即使是那些仍然存在的资产，其去中心化的承诺也只是一个幌子。整个生态系统几乎完全依赖于中心化的市场 OpenSea。OpenSea 使用标准的 Web 2.0 服务器来缓存数据和显示图像。如果 OpenSea 决定封禁某个 NFT 集合，那么该资产实际上就会从用户的视野中消失。你或许仍然持有区块链上的加密收据，但如果没有集中式界面来展示和交易它，这种所有权实际上毫无价值。

同样的，MetaMask 是使用最广泛的 Web3 服务接入点，它并不直接连接到区块链，而是连接到 Infura 的服务器。如果 Infura 决定屏蔽特定区域的 IP 地址，那么这些用户的 Web3 服务就会直接关闭。

### 3.4 现实资产代币化的悖论

现实资产代币化 RWA（Real World Assets）试图打破数字与实体的边界，其核心概念是将房地产、黄金或国债等具有内在价值的实物资产映射到区块链上，转化为可交易的数字代币，旨在实现资产的碎分化所有权和跨地域的流动性。

然而，这一叙事揭示了 Web3 最深刻的悖论：数字代币无法直接控制物理世界。以房地产代币化为例。想象一个平台将纽约的一栋公寓楼产权分割为数百万个代币出售，声称持有代币即拥有房产份额。但在现实的商业逻辑中，土地和建筑的所有权最终仍取决于中心化政府的土地注册处。无论链上的代币如何流转，如果当地法律不承认区块链记录的绝对效力，代币持有者在物理世界中就没有任何合法的求偿权。一旦代币发行方破产跑路，或者实物房产因纠纷被法院查封，投资者手中的代币就仅仅是一串毫无意义的数字。

RWA 并未移除中介，反而增加了一个复杂且昂贵的技术层，最终仍需回归到它试图取代的传统法律与行政体系中寻求庇护。

## 4 共识的尺度是根本

通过对加密货币、区块链和 Web3 这三层营销外衣的解构，我们可以清晰地看到这个俄罗斯套娃背后潜藏着一个共同的逻辑困境：去中心化全体共识与安全和效率之间的权衡。

比特币之所以能够成为这个领域的金标准，是因为它在货币这一特定资产上成功地达成了全球共识，即便这种共识是以极高的能源消耗和极低的底层效率为代价的。作为全球通用的主权货币，它必须追求极致的安全与去中心化。

然而，过去十年的各种衍生泡沫揭示了一个残酷的常识：绝大多数互联网应用并不需要这种全球范围内的全体共识。为了让一个点赞、一次供应链记录或者一段代码的执行去惊动全球数万个节点达成一致，在物理和经济逻辑上都是极其荒谬的。这种对全体共识的迷信，不仅让 Web3 应用在效率上远逊于传统 Web 2.0 系统，更迫使它们为了追求基本的可用性，不得不重新套上中心化的面具，沦为更昂贵、更不透明的科技黑箱。

好消息是，全体共识并不是通往数字主权的唯一路径。只有像全球通用货币这样对信用等级有着严苛要求的系统，才需要那种沉重而绝对的安全边界。对于社交互动、日常金融交易、分布式计算以及内容存储等绝大多数场景，共识的尺度本应是局部的。在这些领域，我们完全可以在局部共识的基础上，通过个人间的信任网络来提高运行效率并保障必要的安全。

正如本书前面章节所介绍的，主权个人系统 SovinDS 以及通过个人信任网络建立的主权个人互联网 SovinWeb，正是这种共识回归常识的体现。它不追求让每一个无关的节点都去验证你的一举一动，而是主张在个人掌握数据主权的前提下，建立起灵活的点对点信任链路。这种基于局部共识的架构，不仅在效率上可以与现有的中心化互联网并驾齐驱，更在隐私保护和抗审查性上提供了质的飞跃。SovinWeb 并不是要重复制造另一个虚幻的技术套娃，它是对现有互联网架构的一次深思熟虑的补充。

当人们剥开那些华而不实的营销名词，真正开始关注个人主权与信任网络的根本规律时，一种更加务实且强大的数字文明才真正开始萌芽。
