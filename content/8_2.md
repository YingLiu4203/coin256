# 8-2 普通人的个人主权

斯诺登在爆料之后的生活更像是一种无可奈何的选择，他可能既不喜欢俄罗斯也不喜欢只能接受比特币和大零币，毕竟现阶段以美元为代表的法币在易用性和流动性上仍占主导地位。实际生活中有很高的道德准则同时有硬刚美国的勇气的人还不多见，要不然这个世界会美好很多。但是个人在不直接与帝国冲突的情况下仍然可以做很多事情。长期来看，除了越来越没有可能也无必要的大规模战争组织，技术的发展让个人或小规模自组织群体可以完成几乎所有的传统国家功能。2008 年，一个匿名叫中本聪的个人或极小团队开发了比特币，打破了国家对货币这种最重要金融工具的绝对垄断。这只是众多个人或小团队可以改变世界格局的一个突出例子。这样的例子还有很多，下面的二个例子代表了二个极端的场景：基于个人的全球协作和完全独立的单人企业家。

## 1 基于个人的全球协作

如果问拥有 22 万员工的微软公司最重要的产品是什么，为其带来几十年垄断地位的 Windows 操作系统应该是不二之选。相比比特币这种一个人可以完成的小软件系统，操作系统作为所有计算机的运行底座，技术含量极高，研发投入巨大。按常理，应该只有 IBM、微软和谷歌这种财力雄厚、组织严密的超级大公司才有能力研发并维护如此规模的产品。然而，林纳斯·托瓦兹（Linus Torvalds）当年仅仅出于个人兴趣就研发了 Linux。从最开始的一个人，到现在通过松散的开源协调机制组织起遍布全球的成千上万名义务开发者，在没有庞大严密组织、仅靠个人情怀与企业赞助的情况下，林纳斯通过一己之力彻底改变了全球操作系统的格局。无论是产品质量、安全性还是使用数量，Linux 都早已领先微软的 Windows。Linux 运行在全球绝大多数的服务器与超级计算机上，还通过 Android 系统统治了数十亿移动设备。而且随着更多人贡献代码和使用 Linux，其在安全性、性能等方面领先 Windows 的程度还在扩大，同时在易用性方面的差距在逐步缩小。

Linux 的开发故事里还有一个有趣的插曲凸显了个人创造力带来的巨大影响。一个像 Linux 这样的复杂软件系统需要全球成千上万人多年几乎不间断的密切协作，其中最核心的也是非常困难的任务就是管理数量庞大（从第一版的一万行到现在的数千万行）的程序源代码。2005 年之前，林纳斯一直采用一个商业版的名为 BitKeeper 的分布式版本控制系统。BitKeeper 的技术虽然是去中心化的分布式系统，即任何感兴趣的开发者都有项目源代码的本地复制，但是作为商业公司的产品，其所有权与控制权是集中的。2005 年 4 月，因为一些版权纠纷，BitKeeper 的所有者收回了 Linux 社区的免费使用权。这是一个屡见不鲜的场景。面对这个突如其来的打击，林纳斯做了一个顶级技术极客的自然反应：既然不让我用，那我就自己写一个。作为行动的第一步，林纳斯对当时市面上所有的开源版本控制系统（如 CVS、Subversion）进行了评估，最后得出了一个非常刻薄但准确的结论，这些全都是垃圾。因为它们都是集中式的，在全球数千人协作时慢得像乌龟，而且极易因为中央服务器宕机而导致整个团队停工。第二步就是动手设计、开发去中心化的版本管理系统，取名 Git。在英国、爱尔兰等地的俚语中，Git 是一个贬义词，意思是“傲慢自大的人”、“讨厌鬼”、“蠢货”或“饭桶”。林纳斯曾开玩笑地解释过为什么选这个词：“我是个自负的人，所以我用自己的名字给我的所有项目命名。第一个是 Linux（Linus's Minix），第二个就是 Git（讨厌鬼）。”林纳斯闭关了大约两周。第一周，他用 C 语言写出了 Git 的核心架构；第二周，Linux 内核的第一个由 Git 管理的版本成功提交。Git 采用的是去中心化的分布式架构和强大的自动合并（Merge）算法，允许全球成千上万名开发者在同一时间、修改同一个软件系统的不同部分。它把软件开发的效率从一条单行道变成了百车并行的超级高速公路。Git 还采用哈希算法保证了数据的绝对完整性。任何代码一旦提交，就几乎不可能丢失或被悄悄篡改。Git 为所有的代码改动产生一个新的快照（Snapshot），保留了项目从诞生第一天起的所有历史记录，这让开源项目在面对成百上千陌生人的代码贡献时，拥有了极高的容错率——写错了用旧的快照重做就是，这极大降低了信任成本。Git 的出现，彻底重塑了人类大规模智力协作的形态，在全球开发者中使用率高达 93%。Git 的重要性与影响力可能不亚于 Linux，怎么强调都不为过。不同于万人参与的 Linux，Git 的设计和核心功能实现是林纳斯用一己之力完成的。

在整个 Linux 和 Git 的开发和演进过程中，国家各级政府能起的唯一作用就是不添麻烦。而讽刺的是，政府连这点都做不到！由于 Linux 基金会（Linux Foundation）注册在美国，美国政府经常对开发者之间的跨国技术交流提出各种基于制裁的限制。美国政府对加密软件有严格管控，而加密和数据安全恰恰是 Linux 的基本核心功能。加州 2025 年出台的《数字年龄保证法案》（Digital Age Assurance Act），强制要求开源社区去核验用户年龄，这意味着 Linux 社区必须建立庞大的中心化身份数据收集系统，这彻底违背了开源、匿名和隐私保护的初衷。政府可能没有搞明白的问题是，如果开发者之间全部采用端到端匿名加密通信，Linux 作为开源免费分发的软件代码，政府的这些法规在宪法保护言论自由的前提下，根本没有执行的抓手和可能性。无论如何，和比特币一样，开源系统赋能个人这个主题会是政府与主权个人之间的一场长期博弈焦点。

## 2 单人企业家（Solopreneur）

如果说 Linux 展现了成千上万个独立个体通过开源协议进行大规模自组织协作的力量，那么这些年兴起的单人企业家（Solopreneur）则向我们展示了主权个人在商业与生活层面的另一种极致形态：原子化个体对自己生活和工作的绝对掌控。彼得·莱弗尔斯（Pieter Levels）是这种人的代表。他是一位在荷兰长大的单人企业家，独自开发和运营十几个高度自动化的互联网产品，年收入超过 300 万美元，且几乎都是净利润和（更加重要的）个人所得。作为独立软件开发者（Indie Hacker），他没有雇佣任何全职员工，也没有固定的办公室，所有的客服、服务器运维、甚至部分辅助开发工作都靠软件工具和 AI 自动化完成。从 2014 年大学毕业开始，他就提着一个背包和一台笔记本电脑，以数字游民身份在亚洲和欧洲各国流动居住。彼得大学学的是管理学，他的软件编程知识完全是利用互联网资源自学而成。他最受欢迎的两个赚钱产品，也是从自己作为数字游民的切身需求出发建立的网站：一个是数字游民榜单 [Nomad List](https://nomads.com/)，列出全球数字友好城市的网速、物价以及签证信息；另一个则是远程工作招聘网 [Remote OK](https://remoteok.com/)，让企业发布各种远程工作职位。

当 2022 年 AI 大模型爆发的时候，他迅速自学了相关知识，推出了几个爆款 AI 应用。比如 Avatar AI 让用户上传自己的照片，通过 AI 生成各种场景的艺术照片或证件照片；Interior AI 则根据一张房间照片，瞬间生成不同风格的室内设计效果图。彼得的成就很大程度上得益于开源代码、云计算、以及 AI 等各种信息技术的普及，让传统的资本和劳动力红利在技术面前被极度压缩，使得一个人就能成为一条完整的产业链。

互联网产品众多，如何赢得用户的信任获得市场是个关键问题。彼得没有销售团队，更没有公关部，他的营销方式就是公开一切 -- 从产品的源代码、服务器运行状态到每月的财务收支，他打造了极度透明的个人品牌，从而赢得了全球用户的信任。作为数字游民的最大好处就是地理套利（Geographic Arbitrage）：可以选择在自由度高、税收与生活成本低、治安良好的地方生活。他在物价极低的泰国清迈、印度尼西亚巴厘岛或者方便舒适的葡萄牙，赚着来自全球（主要是欧美）用户的美元，通过互联网将自己的服务卖给全世界，同时将大部分收入收为己有 ！！！最后这点足以让缴税过半的硅谷程序员羡慕不已了。

彼得曾公开分享过自己的税收与金融交易策略。他的公司设在对海外收入收税极低的新加坡。新加坡只对汇入的海外收入征收 17% 的所得税，不汇入的海外公司收入和个人收入则完全免税。彼得的个人税务居民身份则落在葡萄牙，在搞清楚葡萄牙的签证和税务规则的同时，还顺手做了一个帮助其他数字游民落地葡萄牙的工具项目。他的所有项目都通过 Stripe 接受美元和欧元的网上支付，所有收入与支出也以电子化方式自动转给专业的财务外包公司处理。这种复杂的跨国设置（新加坡公司、葡萄牙个人居民、全球客户以及频繁的亚洲差旅）如果由传统银行来处理各种外币兑换和转账支付，业务会非常昂贵且低效。跨境支付巨头 Wise（前身为 TransferWise）看到了这个巨大的痛点市场，推出了无国界的清算服务 Wise 商业账户（Wise Business）。企业或个人只需通过线上验证，Wise 会瞬间为你生成 11 个主要国家或地区的本地银行账户信息（包括美国的 ACH 路由号和账号、欧洲的 IBAN、英国的 Sort Code、新加坡的 SGD 账户、中国香港的 HKD 账户等）。所有这些地区的资金收付都变成了免费的本地行内转账，资金可以即时或短时间内到达，彻底避免了传统跨国 SWIFT 转账长达数天、动辄几十或几百美元的手续费。通常传统银行的外币兑换会有高达 2% 到 5% 的隐性手续费，Wise 则承诺以极度透明的国际中间汇率，仅收取 0.33% 到 0.8% 的极低手续费来兑换 48 种以上的货币。而最让数字游民或跨国自由职业者喜欢的是，这一切服务都可以通过 API 代码自动执行。

当然，提供全球最强大的法币跨境清算工具，也有法币体制固有的代价。Wise Business 是一家受到各国金融管理局高度监管的传统金融科技公司，其系统出于合规要求，不支持比特币或其他去中心化加密货币的充值、提现、存储或直接支付。深谙国家与法币弊端的彼得自然不会止步于此。作为年入数百万美元的主权个人，彼得曾明确表示自己长期持有比特币。为了对抗法币不可避免的通胀贬值和保护个人财产主权，他会定期将法币收入转换成比特币进行冷钱包长期存储。补充一句，他也同本书的看法完全一致，认为绝大多数所谓的山寨币都是毫无价值、纯粹用于炒作投机的垃圾。

彼得的故事揭示了去中心化的生活是一个缓慢发展的过程。在很长一段时间里，个人的工作与生活会是去中心化行为与中心化机制的混合方式，同时传统的中心化企业比如斯诺登使用的 Substack 内容发布平台和彼得使用的 Wise 资金清算服务也在转型为让个人拥有更多的自主权与灵活性。让人感到欣慰的是这种转变是一个加速的过程。彼得的生活方式也是一个很多人可以效仿的模式，无论你是有专业技能的会计师、设计师、程序员、还是英语老师等，都可以从他的故事里看到各种可能性。
