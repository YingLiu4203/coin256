# 序言：复杂世界的简单支点

> 也许在你身边的人会认为你很古怪甚至是社会的危险分子，但你不会在乎。只要你已经品尝了真理，就绝对不会想要回到无知的状态。
> -- 柏拉图 《理想国》

本书的目的是改变你的世界观和增强对自己生活的控制力。日渐数字化的世界未来无法预测，但今天的信息技术足够让你主动创造属于自己的未来。

1997 年秋天，我在南加州大学（University of Southern California）计算机系求学时，有幸聆听了莱斯利·兰波特（Leslie Lamport）的一场演讲。兰波特是计算机科学界的泰斗，以严谨的逻辑和对分布式系统的深刻洞察著称，他于 2013 年获得了图灵奖。那次演讲的具体技术细节已随时间模糊，但他开场时那番带着戏谑的牢骚，却是一个顿悟，长久地留在了我的记忆里。

在那次演讲中，兰波特半开玩笑、半认真地对台下计算机科学领域的专业听众们说，全世界的计算机科学家都应该感到羞耻。因为改变了人类文明进程的万维网（World Wide Web），并不是由某个顶尖的计算机科学家发明的，而是出自一位名叫蒂姆·伯纳斯-李（Tim Berners-Lee）的英国物理学家之手。兰波特调侃道，当伯纳斯-李在 1989 年发明 Web 时，他并没有创造什么惊天动地的新算法或发明新的硬件设备，所有的底层技术，无论是互联网（Internet）的基础设施，还是超文本（HyperText）的概念，其实早就静静地躺在那儿了。伯纳斯-李所做的，只是设计了一个简单的、通用的通信协议（HTTP）。这个协议是如此简单，任何一个计算机系的本科生都能轻易读懂，要是写成学术论文投稿的话，大概率会因为缺乏技术深度而被拒稿。

然而，历史的有趣之处正是在这里：正是这个低技术含量的简单协议连接起全人类，彻底重塑了我们生活、工作和思考的方式。这个故事揭示了一个关于变革的深刻真理：伟大的变革很少依赖于单点的突破发明，它更像是一场关于组合的艺术，将那些早已散落在各处的现有技术，以一种前所未有的逻辑缝合在一起。正如万维网之于互联网，巨大的变革往往只需要利用已有的技术积木来搭建起一个全新的文明层。当技术的基础设施已经就绪，社会往往只需要一个简单的支点，就能撬动和改变人类的组织与生活方式。

这就是本书的主题：拜自然法则与信息技术所赐，这次变革的支点异常简单，结果却最具颠覆性。信息技术让个人拥有了史无前例的掌控自己通信自由与财产自由的能力，世界逐步进入由主权个人主导的组织形式，而国家对个人的监控力则会逐渐消亡。作为身处这个巨变中心的亲历者，我试图用浅显易懂的语言和一些具体实例描述这次变革背后的信息技术和人类生活演化逻辑，只需要小学毕业的数学与语文理解力就可以明白那些近乎魔术般的信息技术概念与看似复杂的社会变迁现象。

当意识到可以有一种不同生活方式的刹那，你的世界就已经完全不同了。
