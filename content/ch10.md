# 模仿游戏：加密货币、区块链和 Web3

> 无知就是力量。
> 乔治·奥维尔 《1984》

有了基本的比特币常识，可以看到 SovinWeb 的设计是利用比特币的去中心化与开放性让个人拥有货币主权。在此之上，利用 SovinWeb 的安全、抗审查通信能力人们可以开发和使用各种去中心化的金融和非金融应用。这里有必要澄清一个概念：主权个人互联网 SovinWeb 不是 Web3， 它与 Web3 有着本质区别。Web3 其实是一个三层俄罗斯套娃：给比特币套上加密货币、区块链和 Web3 这三层外衣，利用比特币的理念玩一场吸引用户、拓展市场的模仿游戏。

## 1 看懂模仿游戏的必要性

在数字创新史上，鲜有发明能像比特币一样精巧而又具有颠覆性。比特币解决了困扰计算机科学家数十年的难题：如何在没有可信任第三方的情况下创造数字货币稀缺性。它建立了一个无需许可、抗审查、且由数学而非政治驱动的点对点电子现金系统。然而，随着比特币的轰动性成功，行业内也笼罩在一片语言迷雾之中。一些新术语，特别是“加密货币”、“区块链”和“Web3”被创造出来并迅速普及。虽然这些术语通常被认为是比特币技术的自然演进，但仔细审视就会发现，它们不过是营销手段的产物。

要驾驭比特币带来的崭新数字货币世界，首先必须解构用来描述它的语言。这并非仅仅是语义学上的学术探讨，它之所以成为至关重要的防御机制，原因有三：

- 保护财产：投资者经常被营销术语误导，以去中心化资产的名义诱骗用户购买中心化负债。理解提供真正所有权的协议与仅仅提供企业数据库访问权限的协议之间的区别，往往决定着财产主权与破产之间的区别。
- 技术素养：区分去中心化的革命性方案带来的安全与效率的权衡，对于评估所谓“改进功能”至关重要。
- 维护使命：无需许可、抗审查的货币理念正被企业利益集团积极削弱，这些企业试图重新引入比特币原本想要消除的中介机构。将工具（区块链）与目标（自由）混淆，就等于在战争开始之前就输掉了这场战争。

下面可以看到：“加密货币”、“区块链”和 “Web3” 不是技术类别，而是在理念模仿比特币的三个环环相扣的包装词汇，是旨在将比特币的声誉转移到眼花缭乱的产品上的营销术语。绝大多数的产品从根本上不符合比特币的核心价值观。

## 2 “加密货币”的谎言

“加密货币”一词或许是这些概念中最普遍也最容易误导的。要理解个中缘由，首先必须认识到比特币的独特地位。比特币是第一个真正去中心化的数字货币。它无需中央服务器就解决了双重支付问题，创造了一种无法被扣押、无法被审查且对任何人开放的数字资产。这一突破赋予了它巨大的价值潜力，不仅可以作为支付渠道，还可以作为比黄金更灵活的个人主权价值储存手段。随着比特币的崛起，成千上万的模仿者涌现出来。这些项目比如莱特币、狗狗币以及后来成千上万的其他项目都试图分一杯羹，从比特币创造的巨大价值中获利。然而，它们面临着一个品牌定位的问题。如果它们自称为“支付网络”，就会被拿来与 Visa 或 PayPal 比较（而且会落败）。如果它们承认自己是中心化的初创公司，就很难获得用户而且会像证券一样受到监管。它们需要一种方法来证明自己与比特币的等同性。他们需要一个类别，将比特币仅仅视为众多成员之一。

### 2.1 “更好”的比特币

“加密货币”一词便应运而生。中本聪在最初的白皮书中从未使用过“加密货币”一词；他使用的是“电子现金”。该术语最早出现在互联网论坛和邮件列表中，用来描述采用加密机制与去中心化算法的数字货币。虽然它最初可能只是一个自然的描述性标签，但很快就被业界滥用。通过将自己归入“加密货币”的范畴，中心化项目可以暗示它们拥有比特币的基本属性。这使得它们可以把自己包装成“更快的比特币”或“功能更丰富的比特币”。这是一种类别错误。它试图将比特币归类为众多同等资产中的第一个，暗示存在更多新的“更好的比特币”，而比特币就像福特T型车：老旧、缓慢，注定会被更快、更炫酷的新车型所取代。

这种分类理论上没有问题，可是实际上几乎所有新加密货币都忽略了被称为“区块链三难困境”的基本工程现实。在分布式系统中，存在三个相互竞争的目标：去中心化、安全性和可扩展性。一个系统通常只能以牺牲一个目标为代价来优化其它两个目标。比特币的设计明确地优先考虑了去中心化和安全性。它牺牲了底层可扩展性（吞吐量），以确保任何拥有基本计算机的人都可以运行节点并验证网络，从而保持其抗审查性。这种平衡并非偶然，对于一个全球通用的个人主权数字货币而言，这是数学上的必然要求。Hashcash（中本聪为比特币借鉴的工作量证明系统）的发明者 Adam Back 是比特币原始白皮书中引用的少数几位密码学家之一。2013年，在比特币推出四年后，Adam 花费很长时间试图改进比特币的设计。作为一位顶尖的密码学家，巴克力求在不损害其他支柱的前提下，调整变量以提高可扩展性或隐私性。许多尝试之后他得出结论，比特币存在于一个“狭窄的设计空间”中。这意味着该协议经过了高度优化，几乎任何改进某个变量（例如区块大小或速度）的尝试都会对另一个变量（例如用户自托管节点的能力）造成灾难性的削弱。比特币的“慢”是一种特性，而非缺陷，这是确保没有任何单一实体能够控制账本的代价。

### 2.2 作弊史

大多数被冠以“加密货币”之名的资产实际上并没有“解决”这一三难困境，它们只是在作弊。它们通过牺牲去中心化来实现高交易速度（可扩展性）或“灵活性”，同时还利用比特币的营销语言来掩盖这些权衡取舍。比如以 Solana 为代表的加密货币经常被宣传为“更快的比特币”，其交易速度高达每秒 65,000 笔交易 (TPS)。对普通投资者来说，这似乎是一项技术突破。但实际上，这是一种蛮力取舍。为了实现如此高的速度，Solana 要求其验证节点（运行网络的计算机）配备工业级硬件：24 核 CPU、512GB 内存和千兆互联网连接。这些要求使得普通用户根本无法运行节点。因此，该网络由少数富裕实体在数据中心运行服务器来维护。如果政府想要关闭 Solana，他们只需要传唤少数几家云计算服务商。相比之下，一个比特币节点可以在委内瑞拉地下室里一台价值 50 美元的树莓派上运行，可以完全脱离中心化的信息系统服务商。

比特币并不是停滞不前的。中本聪隐身之后比特币生态逐渐形成了一个去中心化的的治理机制，即前文提到的 BIP。 BIP 为比特币提供了改进与发展机制，在不牺牲去中心化与安全的前提下，把比特币交易速度提高了近一倍，更为关键的是为第二层的高速交易应用铺平了道路。BIP 是一个缓慢的、共识驱动的过程，需要节点、矿工和开发者的压倒性共识。相比之下，而大多数“加密货币”的运作方式却像科技初创公司。它们有“基金会”、首席执行官、市场营销预算和发展路线图。它们使用“加密货币”这个隐含去中心化的术语来掩盖这样一个事实：大型风险投资公司（在预售阶段低价购入代币）掌握着大部分投票权。这并非去中心化货币，而是伪装成协议的股东资本主义。一个简单的事实可以说明这个问题：很多“加密货币”引入了“预挖矿”的概念。在公开发行之前，将初始代币供应量的很大一部分分配给内部人士或相关投资者。这与比特币形成了鲜明对比，比特币没有预挖矿。中本聪与任何想参与的人一起公开挖出了第一批比特币。

当然，更能直接撕破这些”加密货币“伪装的事实是，出于各种原因，这些加密货币通常有硬分叉改写交易历史或具有冻结客户资金的经历。比如以太坊采用硬分叉来挽回黑客攻击造成的损失。而类似 USDT 或 USDC 这种打着“加密货币”称号的稳定币内置了运营者可以操控的客户黑名单。和比特币的个人主权原则完全背道而驰。

这也是为什么“加密货币”这个词直到本章才出现的原因，避免用这个极具误导性的词汇来归类比特币。“数字货币”是个更为恰当的词代表一大类在互联网上运行的的各种货币。除了比特币本尊外，其它数字货币都是假“加密货币”之名寄望以貌似比特币的面目获得更大的市场。在清楚地说明了其他数字货币和比特币的根本区别之后，后面关于山寨币的讨论有时会沿用”加密货币“这个词以方便讨论，毕竟这些山寨币的文档以及被误导的普通人都习惯了用这个词。

在面对比特币之外的其它数字货币的灵魂一问是：在可以改写历史或冻结用户资金的加密货币里，个人主权还存在吗？

## 3 “区块链”的谬误

如果说“加密货币”淡化了比特币的去中心化内涵，那么“区块链”一词则淡化了比特币是去中心化的数字货币的基本属性。当人们说区块链时，本质上是想传递一个错误的事实：用区块链作为数据结构的公共账本技术可以用于货币之外的各种应用，比如金融交易、供应链甚至物联网，这样新的应用也具有类似比特币的去中心化与透明性光辉。其实中本聪在其最初的比特币白皮书中从未单独使用过“区块链”一词。他指的是“一系列区块”。这个细节没有那么重要，区块链说法的根本谬误在于除了通用的货币，很少有其他应用需要全体共识，而安全的全体共识带来的低效率（每秒 7 个交易）也不满足多数应用的实际需求。经过十多年的实践与市场验证，所有的区块链应用都有二个结局：要么失败，要么打着区块链的旗号实际用中心化的实现技术来维持运作。有鉴于市场上太多的非货币区块链应用，这里再强调一次，这是所有的区块链应用的二种结局，无一例外。用第一性原理来检验一下，背后的道理很简单：除了货币，还有什么事物需要所有参与者的一致共识？实现全局一致共识的成本极高，对于不需要这种共识的事物，用中心化的实现方式更加简单和高效。如果比特币采用中心化的实现方式，一个有宽带互联网连机的普通个人计算机就足够处理区区每秒数笔的比特币交易了，当然运行成本比运行比特币网络要低一千万倍或更多。不算设备投入，2025 年仅比特币网络的挖矿耗电量就约占全球总电量的千分之五，相比之下，个人高端游戏主机的数百瓦峰值功耗大概是比特币挖矿耗电量的三千万分之一。

“区块链”一词在2015年左右开始流行，成为继山寨币之后推广所谓“去中心化金融 DeFi （Decentralized Finance）“ 应用的市场术语，是企图搭比特币红利便车的各种新兴金融科技企业的营销策略核心。后来又蔓延到非金融领域的供应链管理、公司治理、电子游戏、共享经济等更多领域，毕竟去中心化与透明性的愿景有巨大的市场号召力。对区块链的推销力度，从[区块链革命](区块链革命) 这本充满“想象力”的书的流行程度可见一斑。该书得到许多技术专家和学者大力吹捧，仅举几例如下：

- 史蒂夫·沃兹尼亚克（Steve Wozniak），苹果公司联合创始人： “这本书真是精彩绝伦！其视野之广阔、思想之深刻令人叹为观止。它让我意识到，我们正处于科技、经济和社会发展史上一个前所未有的时代。“
- 克莱顿·克里斯坦森（Clay Christensen），哈佛商学院工商管理教授，《创新者的窘境》作者：“塔普斯科特夫妇撰写了一本关于如何在下一波技术驱动的颠覆性变革中生存并蓬勃发展的权威著作。这本书很可能成为我们这个时代的标志性著作之一。”

这本书是基于多伦多大学四百万美金研究项目的成果，书中列举了许多有重大价值的各种应用：包括金融领域的八个核心功能、公司治理的核心架构、七种新的商业模式、以及十二种颠覆性的万物账本应用。如果觉得这还不够，还有三章分别谈到了经济、政府以及文化的各种革命性应用。十年时间过去了，现在可以看看区块链在 DeFi、传统金融服务和供应链这几个最有直观应用价值领域的实际效果。

### 3.1 去中心化金融 (DeFi)

DeFi 的承诺是建立一个没有权威中心的金融系统：一个无需信任的基础设施，代码即法律，权力分散在多数人而非少数人手中。然而，对该领域的深入剖析揭示了一个与宣传截然不同的现实。在人为炒作和“虚假交易”的推动下，DeFi 生态系统的大部分实际上并非去中心化。相反，它只是一个去中心化的舞台，像 [Hyperliquid](https://hyperfoundation.org/) 这样的协议以中心化银行的不透明性运行，却缺乏监管保障，往往由少数拥有“上帝模式”权限的管理员密钥控制。Hyperliquid 将自己定位为一个高性能的区块链，专为高速交易而构建，号称拥有中心化交易所 (CEX) 的速度和去中心化交易所的自托管能力。其原生代币在 2024 年末和 2025 年引发了大规模的零售抢购热潮。然而，这种“类似中心化交易所”的速度所带来的架构代价是共识机制的严重中心化。Hyperliquid 依赖于一个规模很小而且受控的验证节点，通常只有区区几十个。这种结构与其说是一个去中心化网络，不如说是一个由联盟控制的分布式数据库。Hyperliquid 的口号是”区块链将成为所有金融的归宿“，这种“表演”掩盖了一个关键的事实：当一个网络依赖于少数验证者时，它很容易受到串谋或胁迫的影响。该协议实际上更像是一个专用的服务器集群，而不是一个具有弹性、抗审查性的公共产品。这里的“去中心化”很大程度上只是表面上的，它给人一种使用区块链的感觉，但实际上保留了传统金融科技公司的控制结构。如果服务器集群还不够直观，Hyperliquid 连系统代码都不是开源的，这种做法击穿了最基本的信任底线。即使使用开源代码，大多数 DeFi 协议也并非不可篡改的代码。它们是可升级的软件，由一小群开发者控制。这种控制通常通过多重签名钱包来实现，少数人（通常是 5 个签名者中的 3 个）可以随意更改智能合约逻辑、暂停提现或转移资金。

DeFi 基础设施是中心化的，在区块链的名字掩护下靠人为炒作来制造市场繁荣的气氛。“炒作”不仅仅是一个股票代码，它更是整个行业的经济引擎。来自 Chainalysis 等区块链分析公司和学术研究人员的真实数据，描绘了一幅令人震惊的“虚假交易”图景，这种行为是指买卖资产给自己制造交易量的假象。据估计，2024 年和 2025 年，去中心化交易所 (DEX) 的交易量中，高达 25 亿美元为虚假交易。Hyperliquid 等平台或各种“积分计划”上的大量交易量并非源于自然需求，而是“女巫”行为，用户创建数千个钱包，进行自我交易以获取空投。这造成了一种“鬼城”效应，高额的总锁定价值 TVL (Total Value Locked) 掩盖了缺乏真正的经济效用。多数 DeFi 项目还发行流通量低（低流通量）但完全稀释估值高的代币。这使得内部人士和风险投资机构能够虚增账面价值，用所谓新技术范式炒作蒙蔽的散户投资者。而实际上，这种新技术范式的核心只不过是一个高速的集中交易数据库而已。环顾全球互联网，迄今还是没有一个大规模的去中心化的金融应用。将来如果有，也是小范围的低频交易，逐利的商业公司不会有兴趣去开发运营这种没有油水的金融应用。

所以说 DeFi（去中心化金融）目前的版本是名不副实。它实际上是分布式金融科技，而非去中心化金融。像 Hyperliquid 这样的平台表明，市场更看重速度和投机（炒作），而非抗审查的基本原则。虽然精明的投机客确实可以通过驾驭这种混乱局面获利，比如利用套利机会、参与空投、以及驾驭低流通量代币的波动，但这种成功的投机不应与参与真正去中心化的系统混为一谈。这些参与者只不过是在利用有利可图的混乱局面。当系统保留管理员密钥、依赖少量的受控验证节点并依靠虚假交易量维持运营时，它们并非在构建新的金融系统，而只是在重建旧系统，但问责机制更弱，系统性风险更大。

### 3.2 改造传统金融服务

大约在 2015 年，一个富有号召力的“区块链，而非比特币”叙事轰轰烈烈地出现了，区块链被包装成可以撼动金融界寻租者地位的一场革命。全球最大的金融机构包括银行、清算所和交易所认为他们可以剥离这项技术中的“无政府主义”元素（去中心化、抗审查性），在提高共享账本效率的同时保留安全性与透明性。十足的想象力，十足地体现了无知就是力量这句话的深刻见解。这种对技术的根本误解导致了长达十年的代价高昂的失败。金融机构的 R3 联盟和澳大利亚证券交易所的 CHESS替代项目这两个著名的例子表明，当中心化实体试图采用去中心化架构时，它们往往会创建成本高昂、效率低下的数据库，而这些数据库并不能解决它们最初的任何问题。

R3成立于2014年，曾被宣传为全球银行业精英的终极联盟。鼎盛时期，R3 拥有超过 100 家金融巨头成员，包括高盛、摩根大通、摩根士丹利和桑坦德银行。其目标是构建一个用于银行间结算的“全球账本”，从而消除对账成本。然而，该项目很快陷入了“联盟悖论”（类似于 TradeLens）。银行之间的竞争异常激烈，它们不愿与竞争对手共享透明的账本。到 2016 年末和 2017 年，该联盟开始瓦解。据报道，高盛和桑坦德银行因股权和治理方面的争议而退出。摩根大通也离开了，转而开发自己的内部工具，因为他们意识到，联盟的行动速度取决于其中最慢的成员。R3 最终发布了一个新系统并明确表示不再称其为“区块链”。该系统没有区块，没有全球数据广播，也没有区块链。它实际上是一个带有公证服务的点对点消息系统，一个复杂的分布式数据库，但完全不是最初承诺的革命性的、无需信任的基础设施。即使如此，基于新系统构建的数字贸易融资网络由于缺乏商业可行性，于 2023 年关闭。事实是，银行并不需要分布式账本，他们需要的是更好的 API 和标准消息格式。

澳大利亚证券交易所 (ASX) 的项目是另一次因无知导致的灾难性失败。2016 年，ASX 宣布了一项雄心勃勃的计划，用基于区块链的解决方案来取代其老旧的 CHESS（清算所电子子登记系统）。该项目原计划于2020年启动，却最终沦为资金和工程时间的无底洞，最终在2022年11月惨遭取消。ASX 被迫注销了近 2.5 亿澳元（约 1.7 亿美元）的税前成本。由于项目失败严重，澳大利亚证券和投资委员会（ASIC）于2024年起诉ASX，指控其在项目暗中失败的情况下，仍就其“进展顺利”的虚假声明误导公众。埃森哲的独立评估揭示了核心问题：延迟和复杂性。区块链系统本身就存在延迟，因为交易必须批量处理并排序才能达成共识。对于需要毫秒级结算的高频股票交易所而言，这种架构是致命的。评估发现，该系统过于复杂，试图将“串行处理”强加于分布式账本，以至于无法管理。更根本的原因在于，ASX 是一家受监管的垄断机构，也是澳大利亚股票交易的中央权威机构。用旨在消除中央权威机构的技术来取代中央权威机构的数据库，这是一种逻辑谬误。该项目增加了去中心化的成本（复杂性、吞吐量降低），却没有带来任何好处（抗审查性），实际上只是重新创建了一个速度更慢的旧版本。

R3 和 ASX 的案例表明，区块链是一种权衡，而非优化。去中心化是以效率为代价的。像比特币这样的公共区块链愿意付出这种代价，以实现抗审查性和独立于国家的货币。然而，像银行和证券交易所这样的中心化机构已经拥有更新账本所需的“信任”。对它们来说，区块链只不过是一个效率低下的数据库。R3 最终解散成为一家软件供应商，而 ASX 则回归了传统的中心化系统，这证明在许多传统的金融领域，共享的电子表格或数据库通常就足够了。

### 3.3 优化供应链

在区块链发展史上，鲜有失败案例能像 TradeLens 那样具有警示意义。TradeLens 于2018年 由 IBM 和 A.P.穆勒-马士基集团合资成立，曾被誉为企业区块链的“标杆之作”。它承诺通过供应链数字化，用不可篡改、透明的账本取代堆积如山的纸质文件，从而彻底革新价值20万亿美元的全球贸易行业。不出意外，该平台在 2022年底关闭了。它的失败并非源于技术缺陷，而是源于区块链谬误，是将去中心化技术强行塞进中心化企业架构的典型案例。

TradeLens 背后的动机是合理的。全球航运业的运作方式向来陈旧落后，仍然依赖提单等纸质文件，提单是一种法律所有权证明，通常需要通过空运才能在船舶抵达之前送达。一次货运可能需要几十个不同机构的审批，并产生数百份通信记录。TradeLens 利用一种需要许可加盟的区块链 Hyperledger Fabric 来创建单一的电子化交易共享视图，同时可以保障隐私。其目标是让承运商、港口、海关和货运代理能够访问实时数据，从而降低摩擦和成本。

该项目的失败根植于其基因之中。TradeLens 并非中立的行业工具；它是马士基（当时全球最大的航运公司）和 IBM 共同拥有的知识产权。这立即造成了利益冲突。TradeLens 的运作需要网络效应，它需要马士基最强劲的竞争对手加入该平台并共享数据。这些竞争对手被要求加入一个受竞争者控制并从中获利的系统。尽管一些竞争对手在经过多年的犹豫和治理结构重组后最终加入，但固有的阻力是致命的。业内人士将 TradeLens 视为“马士基的产品”，而非一个中立的去中心化网络。正如一位供应链高管所言：“我为什么要把我的数据交给最大的竞争对手？”。2022 年 11 月，马士基宣布终止TradeLens项目，理由是未能实现“全面的全球行业合作”和“商业可行性”。虽然该平台已成功吸引了一些海关当局和港口加入，但未能达到支付运营成本所需的承运商和货运代理商的临界规模。其收入模式——对数据访问和数字化服务收费——无法覆盖运行专有企业区块链的巨额运营成本。此外，这项技术本身也可以说是设计不当。TradeLens 实现的大部分功能（数字化文档共享）都可以通过标准的云数据库和已有系统来实现。通过强行采用区块链架构，该项目增加了复杂性和成本，降低了运营效率，却未能提供公共区块链所具备的抗审查性和中立性。TradeLens 的失败为“私有区块链”的叙事敲响了警钟。它凸显了联盟悖论：由联盟控制的区块链往往过于中心化，难以获得竞争对手的信任；同时，与中心化数据库相比，其复杂程度又不足以保证效率。

## 4 Web3 的迷雾

现在来到了俄罗斯套娃（matryoshka）的最外层。套娃中心是微小而坚实的比特币，一种旨在将货币与国家分离的特定功能性发明。包裹着比特币的是第二层的加密货币和第三层的区块链，它们试图将比特币的特定架构推广为一种通用的“技术”。最后，最外层最大、最空心的套娃是 Web3，这是一个旨在将加密货币与区块链赌场包装成互联网必然未来的营销术语。Web3 并非一场技术革命，它是另一场更大的模仿游戏。它模仿比特币的去中心化架构，兜售一种实际上高度中心化、效率低下且依赖于它声称要取代的现有结构的产品。

### 4.1 来历

“Web3” 的历史是一部身份盗窃史。这个术语最初由两位不同的先驱者两次提出，分别用来描述两种截然不同的未来互联网愿景。从 1999 年到 2006 年间第一版的 Web 3.0 是指语义网。早在比特币出现之前，万维网的发明者蒂姆·伯纳斯-李就将 Web 3.0 定义为语义网。他的愿景并非关乎货币或代币，而是关乎数据。在 Web 2.0中，数据是非结构化的，机器无法“理解”一段特定的文本字符串代表的是日期还是价格。伯纳斯-李设想了一个所有数据都相互链接且机器可读的网络，使人工智能代理能够代表我们浏览网络。由于复杂性，这一愿景在很大程度上停滞不前，尽管像 Schema.org这样的技术中仍然存在一些雏形。

第二版的 Web 出现于 2014 年在斯诺登事件之后，以太坊联合创始人加文·伍德重新定义了 Web 3.0 这个术语。伍德将其定义为“安全社交操作系统”，而非语义网。他的目标是抵抗审查。他设想了一个用户无需信任中心化服务提供商（例如谷歌或脸书）来保管其数据的网络。取而代之的是，加密证明取代了“信任”。这一定义为我们今天所看到的行业奠定了基础。到 2021 年迎来了第三版的 Web3 定义升级，原因是“加密”一词已变得令人反感，与勒索软件、价格波动和诈骗联系在一起。风险投资公司 a16z（Andreessen Horowitz）和其他机构积极推广 Web3 一词，将其作为一个更简洁、更友好的品牌重塑。它将叙事重点从“加密货币”与“区块链”的停滞不前和种种丑闻转移到了更加迷人的“拥有互联网”愿景。

### 4.2 Web 3 愿景

Web3的理论目标通常可以用互联网能力的三阶段演进来概括：Web 1.0（阅读）：用户消费由发布商（例如AOL、静态HTML）提供的内容；Web 2.0（读写）：用户创建内容（社交媒体、博客），但平台拥有数据和盈利权；Web3（读写所有权）：用户创建内容，并通过私钥拥有底层资产（代币、NFT、数据）。可以看到 Web3 的主要目标是去中介化。在 Web3 中，“所有权”并非法律合同，而是一种加密现实。Web3 承诺构建一个更加自由的互联网

在 Web3 应用中身份是可携带的可移植的，而不是需要租赁的。在 Web 2.0 中，你的身份是分散的。你拥有 Google 账号、Facebook 账号和 Apple ID。如果 Twitter 封禁了你，你会失去所有关注者和历史记录。在 Web3 中，你的身份就是你的钱包地址（例如，“使用以太坊登录”）。你可以将你的声誉、交易记录和社交关系图谱从一个网站带到另一个网站。新的 Web3 应用可以将你的“关注者”变成一种可移植的资产，它属于你，而不是平台。

Web3 也让个人拥有数字资产。在像 Fortnite 这样的 Web 2.0 游戏中，你花 20 美元购买一个“皮肤”。你实际上并不拥有它；你只是在 Epic Games 的封闭生态系统中租用像素。你不能出售它、借出它，也不能把它带到其他游戏中。在 Web3 游戏中，这个皮肤就是一个 NFT（非货币技术）。如果你退出游戏，可以在二级市场出售皮肤来收回投资。即使游戏开发商关闭服务器，该资产仍然保留在你的钱包中，理论上可以在社区开发的衍生游戏中使用。

Web3 支持创作者经济。在 Web 2.0 时代，音乐人在 Spotify 上每播放一次歌曲只能赚取几美分。而在 Web3 时代，音乐人可以直接将歌曲以代币的形式出售给粉丝。这种代币可以编码可编程的版税。例如，确保每次歌曲被转售时，原作者都能自动获得售价的 10%，永久有效，无需律师或催收机构强制执行。理论上 Web3 允许创作者直接向粉丝出售音乐而无需 Spotify 抽成，也允许用户直接向亲属转账而无需银行收取手续费。

### 4.3 Web3 现实

如果深入剖析 Web3 这一现代术语，就会发现它的结构与俄罗斯套娃相似。它并非单一的整体技术，而是一个层层叠加的结构，外层的 Web3 只是一层谎称拥有核心价值的迷雾。在坚实而小巧的核心，坐落着比特币。比特币提供了一个真正的个人主权货币和交易自由。包裹着核心的是二层套娃：加密货币和区块链。当数字货币的直接模仿者山寨币提供不了有价值的改进而徘徊不前时，金融界和创新者试图将比特币的特定架构通用化。他们提取了“区块链”数据结构，并将其应用于从智能合约到供应链等各个领域。这一层通过模仿比特币的特性（代币、账本）来扩大市场，却不具备其去中心化特性的核心价值。Web3 是外壳上最大、最空洞的一层，其本质上只是一个脆弱的接口，嫁接在老旧的 Web 2.0 基础设施之上。要理解外壳为何如此脆弱，可以分析一下几个著名的 Web3 应用。在所有情况下，这项应用都承诺实现去中心化，但最终却依赖于中心化的守门人。可见 Web3 并非一个全新的互联网，它只是一个蹩脚的插件，仍然完全依赖于它声称要推翻的中心化统治者。

#### 4.3.1 IPFS（存储神话）

星际文件系统 (IPFS) 的设计初衷是取代 HTTP。用户不再需要向特定服务器请求文件（位置寻址），而是根据文件的加密哈希值向网络请求内容本身（内容寻址）。理论上，这使得数据永久存在且抗审查；只要有一个节点持有该文件，它就始终在线。现状：实际上，IPFS 上的文件并不会神奇地永久存在。它们必须由节点“锁定”（托管）。由于手机和浏览器本身并不运行 IPFS 节点，用户需要依赖“锁定服务”。此外，浏览器无法直接读取 IPFS 的内容，因此用户需要通过中心化网关访问内容。所谓的“去中心化”存储实际上只是增加了一些步骤的云存储。IPFS 的架构依赖于中心化的持久化存储和中心化的网关。

#### 4.3.2 MetaMask 和 Infura（访问瓶颈）

MetaMask 是访问 Web3 的标准钱包。其承诺是“自主权”，用户可以直接连接到区块链来广播交易，绕过银行和服务器。实际现状是浏览器扩展程序无法直接与运行区块链协议的网络通信。为了解决这个问题，MetaMask 默认使用名为 Infura 的服务，该服务属于一个商业公司。Infura 就像一个大型电话运营商，在亚马逊云服务 (AWS) 上运行着数千个节点，为数百万用户处理流量。这造成了一个巨大的中心化瓶颈。用户并非连接到一个无需信任的网络。他们信任 Infura 来读取账本。如果 Infura 宕机（这种情况已经发生过），或者他们决定审查某些 IP 地址的交易（就像他们之前短暂地对委内瑞拉用户所做的那样），“Web3”就会直接关闭。去中心化网络几乎完全通过中心化的企业 API 访问。

### 4.3.3 OpenSea 和 NFT（所有权的幻象）

非同质化代币 NFT (Non-fungible Tokens) 承诺对数字资产拥有不可篡改的所有权。与 Epic Games 可以删除的 Fortnite 皮肤不同，NFT 存在于区块链上用户钱包中。但是大多数用户通过中心化市场 OpenSea 与 NFT 进行交互。OpenSea 使用标准的 Web 2.0 服务器缓存数据、显示图像并促进交易。所有权与拥有权的体验是不同的。如果 OpenSea 认为某个集合违反了其服务条款，它会将该资产下架。虽然代币在技术上仍然存在于区块链上，但其图像已从用户的钱包视图中消失，并且对于市场上的绝大多数人来说，该资产将无法交易。这种“不可阻挡”的资产因中心化的用户界面层而变得一文不值，这证明平台仍然掌握着权力，就像在 Web 2.0 时代一样。
